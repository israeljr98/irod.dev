SCHEME INTERPRETER

An interpreter for Scheme, a dialect of the Lisp family of programming languages, written entirely in Python. It is capable of executing a subset of Scheme's functionality, including primitive expressions, user-defined procedures, special forms, and more. The program parses input Scheme expressions and process them through a Read-Eval-Print loop.

technologies used:

Python, Token Parsing

NumC is a matrix operations library for Python powered by C via the Python/C API. This project incorporates data-level parallelism and thread-level parallelism to achieve significant performance increase for heavy matrix computations.

technologies used:

C, Python, OpenMP Library, AVX Intrinsics, Python/C API
